## 📂 Highlight Project

### 🔐 Secure Bank Network using VLANs
Enterprise-style network design for a banking environment with:
- VLAN segmentation for departments (admins, tellers, customers)  
- EtherChannel for aggregation and redundancy  
- STP configuration to prevent loops  
- DHCP for dynamic host addressing  
- ACLs for role-based access control  
- Router-on-a-stick for inter-VLAN routing

**Repository:** https://github.com/aayush-doke/bank-network-vlan
